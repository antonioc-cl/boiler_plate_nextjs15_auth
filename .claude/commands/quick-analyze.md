# @quick-analyze

Quickly analyze requirements for a single feature implementation.

## Usage
```
@quick-analyze
```

## Actions
1. Gather feature requirements
2. Identify technical approach
3. Assess complexity and effort
4. Check dependencies
5. Define success criteria
